# Formulário de Perguntas e Respostas

[![Logo]([images/logo.svg](https://raw.githubusercontent.com/devfelipelimabr/perguntas-game/a611c2555f4e9641b61ccfb1e08b6d57e326a7c7/public/images/logo.svg))](https://raw.githubusercontent.com/devfelipelimabr/perguntas-game/a611c2555f4e9641b61ccfb1e08b6d57e326a7c7/public/images/logo.svg)

Uma aplicação web interativa de formulário de perguntas e respostas.

## Descrição

O Formulário de Perguntas e Respostas é uma aplicação web que permite aos usuários responder a um conjunto de perguntas e receber feedback imediato sobre suas respostas. É uma maneira divertida de testar seus conhecimentos em vários assuntos.

## Dependências

- Node.js (versão 12 ou superior)
- body-parser (^1.20.2)
- ejs (^3.1.9)
- express (^4.18.2)
- mysql2 (^3.2.4)
- sequelize (^6.31.1)

## Instalação

1. Clone este repositório em sua máquina local:

```bash
git clone https://github.com/devfelipelimabr/perguntas-game.git
```

2. Acesse o diretório do projeto:

```bash
cd perguntas-game
```

3. Instale as dependências do projeto:

```bash
npm install
```

## Uso

1. Execute o seguinte comando para iniciar a aplicação:

```bash
node index.js
```

2. Abra seu navegador e acesse `http://localhost:3000` para começar a responder às perguntas.
